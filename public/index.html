📖 Cách Lấy API Key

DeepSeek

  1. Truy cập platform.deepseek.com
  2. Đăng ký / Đăng nhập
  3. Vào API Keys → Tạo key mới
  4. Sao chép key bắt đầu bằng sk-

Gemini 2.5 Flash

  1. Truy cập aistudio.google.com
  2. Đăng nhập bằng tài khoản Google
  3. Nhấn "Get API Key"
  4. Sao chép key bắt đầu bằng AIza
