# DeepSeek UI/UX Skills — Bộ Prompt Template Cho Thiết Kế Web App

Converted từ `ui-ux-pro-max` (200+ quy tắc UX nguyên bản).
Mỗi file là một "skill" độc lập, copy-paste vào System Prompt của DeepSeek để dùng.

## Cấu Trúc

```
deepseek-skills/
├── INDEX.md                    ← File này
├── accessibility-design.md     ← Accessibility (quan trọng nhất)
├── touch-interaction.md        ← Touch target, spacing, gesture
├── animation-motion.md         ← Animation, transition, easing
├── color-typography.md         ← Màu sắc, font chữ, contrast
├── layout-responsive.md        ← Layout, responsive, grid
└── forms-feedback.md           ← Form, toast, validation, dialog
```

## Cách Dùng

1. **Chọn skill phù hợp** với task hiện tại
2. **Copy toàn bộ nội dung** file skill đó
3. **Dán vào System Prompt** khi gọi DeepSeek API
4. Hoặc **dán vào ô "Luật Lệ Cốt Lõi"** trong app game để AI tuân theo

## Ví Dụ

```
Task: "Sửa nút bấm cho dễ bấm trên mobile"
→ Dùng skill: touch-interaction.md

Task: "Làm animation cho modal"
→ Dùng skill: animation-motion.md

Task: "Chọn màu sắc và font cho app"
→ Dùng skill: color-typography.md
```
