# 🤝 Đóng Góp Vào DeepSeek Novel Game

Cảm ơn bạn đã quan tâm đóng góp! Dự án này là một game tiểu thuyết tương tác AI, sử dụng DeepSeek và Gemini API.

## 🚀 Bắt Đầu Nhanh

```bash
git clone https://github.com/hoangdh1012/DEEPSEEK-CHAT-WEB.git
cd deepseek-novel-web
npm install
node server.js
```

## 📋 Quy Trình Đóng Góp

1. **Fork** repo này về tài khoản GitHub của bạn
2. **Clone** fork về máy
3. **Tạo branch mới**: `git checkout -b feature/ten-tinh-nang`
4. **Code** và test kỹ
5. **Commit** với message rõ ràng
6. **Push** lên fork của bạn
7. **Tạo Pull Request** vào repo gốc

## 🎯 Những Gì Có Thể Đóng Góp

- 🐛 **Sửa lỗi** — Báo cáo và sửa bug
- ✨ **Tính năng mới** — Đề xuất và code tính năng
- 🌐 **Dịch thuật** — Dịch UI sang ngôn ngữ khác
- 📝 **Tài liệu** — Cải thiện README, hướng dẫn
- 🎨 **UI/UX** — Cải thiện giao diện

## 📐 Code Style

- JavaScript: Dùng ES6+, `'use strict'`
- CSS: Dùng CSS variables (custom properties)
- HTML: Semantic tags, accessibility (aria-labels)
- Comment tiếng Việt cho logic phức tạp

## ⚠️ Lưu Ý Quan Trọng

- **KHÔNG commit API Key** vào code
- Test trên cả DeepSeek và Gemini trước khi PR
- Giữ backward compatibility
- Một PR = một tính năng/sửa lỗi
